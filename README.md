Đây là một dự án **Next.js** được khởi tạo bằng lệnh `npx create-next-app@latest [project-name] [options]`

## Bắt đầu

Trước tiên, Hãy cài đặt dependencies thông qua lệnh:

```bash
npm install
```

Sau khi cài đặt xong, hãy chạy máy chủ dev:

```bash
npm run dev
```

Mở địa chỉ [http://localhost:3000](http://localhost:3000) trong trình duyệt để xem kết quả.
